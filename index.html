ACTIVE

AI Meeting Summarizer

Automated transcripts and action items for Zoom calls.

Problem

Teams spend hours in meetings, but critical information and action items are often lost or require manual note-taking and summarization, which is time-consuming and inefficient.

Solution

This tool integrates with Zoom to automatically transcribe recordings, identify speakers, and use an LLM to generate a concise, structured summary with clear action items assigned to participants.

Tech Stack

Node.js, PostgreSQL, Redis, Docker, AWS Lambda.

PIVOTED

Dynamic Pricing API

Real-time pricing engine for e-commerce stores.

Problem

E-commerce stores struggle to price products competitively and react to market changes quickly. Manual pricing is slow and often sub-optimal.

Outcome & Pivot

While the API worked, pilot users revealed that their core problem wasn't pricing but knowing *when* to reorder stock. The project was pivoted to focus on an inventory forecasting tool, leveraging the data infrastructure we had already built.

Tech Stack

React Native, Firebase, Google Maps API.

SUNSET

Social Travel Planner

Collaborative itinerary planning for groups.

Problem

Planning a trip with a group of friends is a coordination nightmare, often involving messy group chats, spreadsheets, and lost links.

Outcome & Sunset Decision

The app achieved its goal of simplifying planning. However, the "kill metric" was a D7 retention rate of 20%. After the pilot, retention was only 8%. The user acquisition cost was also projected to be too high for a viable business, so the project was sunsetted.
